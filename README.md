# Multiplayer Snakes & Ladders

## About the Game

A remake of the classic Snakes and Ladders Board game.

### Objectives.

- Teaching the basics of socket programming using **Node.js** and **socket.io**

## Codebase Used

- HTML / CSS / Javascript
- Node.js
- socket.io
